Gedächtnisspiel in JavaScript

Hier der Code für das Gedächtnisspiel.

Der Code wurde im Rahmen einer Schulaufgabe geschrieben.

Die einzelnen Codeteile werden einfach in einem JavaScript Ordner eingefügt. Das Gedächtnisspiel an sich benötigt dann noch eine Seite, in dem es ausgeführt wird. Die entsprechenden Bilder, verlinkungen usw. müssen entsprechend angepasst werden.

JavaScript Gedächtnisspiel game_einfach.js

JavaScript game_einfach als Teil des Gedächtnisspiels dieser Website. Bei ganz unten im Code, kann man die Menge der gezogenen Karten einstellen (die Nummer ändern). Theoretisch kann man das auch beim starten des Spiels auswählen lassen. Da das Gedächtnisspiel auf dieser Website aber Möglichst einfach für meine Tochter sein sollte, habe ich dies mit mehreren Sites und Skripten gemacht. Damals konnte Sie noch nicht lesen und ich konnte so das Spiel einfach öffnen und Ihr zeigen, wie man den Schwierigkeitsgrad einfach auswählen kann.
